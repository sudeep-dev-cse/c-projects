# 🗳️ Online Voting System in C

## 📌 Overview

This project is a console-based Online Voting System developed using the C programming language. It simulates a simple and secure voting process where users can cast their votes, and the system ensures that each voter can vote only once.

---

## 🚀 Features

* 🔐 Voter Authentication (using pre-defined voter IDs)
* ❌ Duplicate Voting Prevention
* 🗳️ Multiple Candidates Support
* 📊 Real-time Vote Counting
* 📈 Result Display with Graph (bar representation)
* 👨‍💻 Admin Access for Result Viewing

---

## 🛠️ Technologies Used

* C Programming Language
* Standard Libraries (`stdio.h`, `string.h`)

---

## 📂 How It Works

1. User enters a valid Voter ID
2. System verifies the voter
3. User selects a candidate
4. Vote is recorded securely
5. System prevents re-voting
6. Admin can view final results

---

## ▶️ How to Run

```bash
gcc c_project.c -o voting
./voting
```
